# MemoryCardGame
记忆卡牌游戏

有三个目录文件：
1. 源码文件：项目的源代码。
2. 记忆大师游戏 setup：windows下的安装程序，可以直接双击进行安装。
3. 记忆大师游戏（已打包）：导入依赖库文件后的程序。
