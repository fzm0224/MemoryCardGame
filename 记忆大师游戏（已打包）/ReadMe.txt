使用Qt制作的简单游戏
使用说明：
打开MemoryCard_exe文件夹，找到MemoryCard.exe文件（电脑在设置显示文件后缀名的情况下才显示.exe后缀名，否者显示为MemoryCard文件），然后双击运行。
也可以在桌面创建快捷方式：
在桌面右击--新建--快捷方式--浏览--选中这个路径下的该文件--下一步--完成

未解决问题：
在Qt编译环境下调试运行时，指定的音频可以正常播放，打包后点击exe文件，音频播放无效。
解决：
两年后，2020.08.08，空闲时间再次看来一下这个项目，突然发现打包后的文件中没有媒体音频文件，然后将音频文件目录music复制到打包的目录下，再次点开可执行程序，音频正常发声。

另外使用Inno Setup这个免费的 Windows 安装程序制作软件，制作了一个setup安装程序MemoryCard Setup.exe，可以通过双击该安装程序进行安装。
